## Installation Guide

### macOS 10.15 Catalina and Later

1. **Download** `LandscapeDesigner.dmg` from GitHub Releases
2. **Open** the DMG file (double-click in Finder)
3. **Drag** the "Landscape Designer" app icon to the "Applications" folder
4. **Eject** the DMG when done
5. **Launch** the app from Applications folder or Spotlight search

### First Launch

On first launch, macOS may display a security warning. Click **"Open"** to allow the app to run.

## Troubleshooting

### App Won't Open
- Try right-clicking the app and selecting "Open"
- Go to System Preferences > Security & Privacy
- Click "Open" next to the app if prompted

### Can't Save Projects
- Check ~/Documents/LandscapeDesigns folder exists
- Verify write permissions on Documents folder
- Try saving with a different project name

### Drawing Not Showing
- Ensure "‚úèÔ∏è Pen Tool" is toggled ON
- Try adjusting line width with slider
- Clear canvas and try drawing again
